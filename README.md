## 📌 Project Description

This project implements a Hidden Markov Model (HMM) using the Baum–Welch Algorithm in C language. The algorithm estimates transition and emission probabilities based on an observation sequence.

---

## 📂 Project Files

- main.c → Runs the program
- hmm.c → Contains HMM and Baum-Welch implementation
- hmm.h → Header file
- README.md → Documentation

---

## ▶️ How to Compile and Run

### Step 1: Open terminal inside project folder

### Step 2: Compile

```
gcc main.c hmm.c -o hmm
```

### Step 3: Run

```
./hmm
```

---

## 📊 Output

The program prints:

- Initial Transition Matrix
- Updated Transition Matrix
- Updated Emission Matrix

These are calculated using the Baum–Welch training process.
